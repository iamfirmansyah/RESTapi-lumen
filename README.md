# lumen-auth-example

## Usage

- `git clone https://github.com/iamfirmansyah/RESTapi-lumen.git`
- `composer install`
- `php artisan jwt:secret`
- `php artisan migrate`
- `php artisan db:seed`
- `php -S localhost:8000 -t public`
- `if error migration, import sql api_yesdok.sql`
- `API Colection "API LUMEN"`
